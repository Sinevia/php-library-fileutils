## Installation ##

Add the following to your composer file:

```json
   "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/sinevia/php-library-fileutils.git"
        }
    ],
    "require": {
        "sinevia/php-library-fileutils": "dev-master"
    },
```
